Classes in Javascript

  • Classes were introduced in EcmaScript 2015 (ES6) to provide a cleaner way to follow object-oriented programming patterns.
  • A class is a blueprint or template for the object. You can create an object from the class.
  • Classes are similar to functions. Here, a class keyword is used instead of a function keyword.
  • The constructor method is called automatically when a new object is created.
  • Always add a constructor() method. If you do not define a constructor method, JavaScript will add an empty constructor method.
  • new keyword: creates an new empty object, takes that object and sets the value of this to be an empty object and then calls the constructor method in bthe class.
