Asynchronous JavaScript code is something, which start at current interval and ends later.

Asynchronous programming is a way for a computer program to handle multiple tasks simultaneously rather than executing them one after the other.

Functions running in parallel with other functions are called asynchronous.

Callback Function:
  • Callback functions are functions that are passed as arguments in other functions.
  • This technique allows a function to call another function
  • A callback function can run after another(first) function has finished execution
  • A callback is simply a piece of JavaScript that gets executed after something else has finished executing.
  • When you pass a function as an argument, remember not to use parenthesis.
    Right: myCalculator(5, 5, myDisplayer);
    Wrong: myCalculator(5, 5, myDisplayer());
