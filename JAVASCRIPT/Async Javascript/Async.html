HTTP request: (Hypertext transfer protocol)

  • We make HTTP requests that can be used to send or receive data from a server side resource
  • Data can be sent using the HTTP POST method and received using the HTTP GET method.
  • HTTP requests in JavaScript are a set of predefined methods used for interacting with data stored on servers.
  • Every request sent to a server includes an endpoint(URL) and the type of request being sent(GET, POST).
  • You can see an endpoint as a gateway between two programsÀê the client and the server.
  • The client is the program that sends a request while the server is the one that receives the request.
  • The server sends back a response depending on the validity of the request.
  • If the request is successful, the server sends back data in either XML or JSON format(JSON in most cases), and if the request fails, the server sends back an error message.
